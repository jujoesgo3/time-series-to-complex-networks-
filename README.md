# Time Series to Complex Networks

Este repositorio contiene el código usado en el trabajo de grado:

**"Análisis Comparativo de Osciladores Clásicos y Cuánticos mediante
Teorı́a de Redes Complejas"**

## Modelos implementados

- Oscilador armónico
- Oscilador de Duffing
- Modelo de Kuramoto

Para cada sistema se realizan:
- Simulaciones clásicas
- Simulaciones cuánticas
- Construcción de redes complejas
- Visualización de redes

## Requisitos

python 3.12  
qutip 5.2.2
scipy 1.16.3
numpy 2.0.2
